# Duckhunt100

Mobile-first duckhunt-app voor de housewarming van Bram en Kayleigh, die per nu naar Brabant zijn verhuisd. Verwacht een genummerde serie van 100 voortvluchtige eenden, worstenbroodwaardige chaos en een botte Brabantse camp-toon met een dikke knipoog. Ook leuk voor hun tweeling: zoeken, aanwijzen, foto maken, door.

## Start lokaal

```sh
python3 -m http.server 4173
```

Open daarna:

```txt
http://localhost:4173
```

## Wat zit erin

- vaste serie van 100 genummerde eenden, elk met eigen naam
- kids-proof activiteit voor de tweeling en andere kleine speurders
- echte foto's van de eendenserie verwerkt als sfeerbeelden
- per eend een eigen initialen-tegel in eendenthema in plaats van 100 keer dezelfde thumbnail
- alle zichtbare voornamen uit de groepsapp-screenshots verwerkt in de namenlijst
- afvinken per eend met kwakbewijs
- voortgangsbalk en unlock-tekst bij 100/100
- namen en notities per eend
- filters voor alle, gevonden en nog zoek
- bewijsfoto's en duckwall met eendenbeelden
- voortgang wordt lokaal in de browser bewaard

De app is volledig statisch en kan dus op vrijwel elke static host worden gezet.
